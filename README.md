## Prerequisites

Before diving into Machine Learning, you should have a basic understanding of:

### Essential Skills:
- **Python Programming**: Basic syntax, data structures (lists, dictionaries), functions, loops
- **Mathematics**: 
  - Basic algebra
  - Basic statistics (mean, median, standard deviation)
  - Basic linear algebra (vectors, matrices) - can learn as you go
- **Data Handling**: Working with CSV files, basic data manipulation

### Don't Worry If:
- You're not a math expert (we'll learn as we go!)
- You're new to programming (Python is beginner-friendly)
- You've never done ML before (that's why we're here!)

## Learning Path

### Phase 1: Foundations (Weeks 1-2)
**Goal**: Get comfortable with Python and data manipulation

1. **Python Basics**
   - Variables, data types, loops, conditionals
   - Functions and modules
   - Working with libraries (NumPy, Pandas)

2. **Data Manipulation**
   - Loading data (CSV, JSON)
   - Exploring data (head, describe, info)
   - Basic data cleaning

**Practice**: Complete 2-3 small data analysis projects

---

### Phase 2: Introduction to ML Concepts (Weeks 3-4)
**Goal**: Understand what ML is and basic concepts

1. **What is Machine Learning?**
   - Supervised vs Unsupervised vs Reinforcement Learning
   - Training vs Testing data
   - Features and labels

2. **Your First ML Model**
   - Linear Regression (predicting numbers)
   - Classification basics (predicting categories)
   - Model evaluation metrics

**Practice**: Build a simple house price prediction model

---

### Phase 3: Core ML Algorithms (Weeks 5-8)
**Goal**: Learn the most important ML algorithms

1. **Supervised Learning**
   - Linear Regression
   - Logistic Regression
   - Decision Trees
   - Random Forest
   - K-Nearest Neighbors (KNN)
   - Support Vector Machines (SVM)

2. **Unsupervised Learning**
   - K-Means Clustering
   - Principal Component Analysis (PCA)

3. **Model Evaluation**
   - Train/Test split
   - Cross-validation
   - Confusion Matrix
   - Accuracy, Precision, Recall, F1-Score

**Practice**: Build 3-5 different models on different datasets

---

### Phase 4: Deep Learning Basics (Weeks 9-12)
**Goal**: Introduction to neural networks

1. **Neural Networks Fundamentals**
   - What are neurons and layers?
   - Activation functions
   - Forward and backward propagation
   - Loss functions

2. **Building Neural Networks**
   - Using TensorFlow/Keras
   - Building your first neural network
   - Training and tuning

3. **Convolutional Neural Networks (CNNs)**
   - Image classification
   - Building CNN models

**Practice**: Build an image classifier

---

### Phase 5: Advanced Topics (Weeks 13+)
**Goal**: Explore specialized areas

1. **Natural Language Processing (NLP)**
   - Text preprocessing
   - Word embeddings
   - Sentiment analysis
   - Text classification

2. **Large Language Models (LLMs)**
   - Understanding transformers
   - Working with pre-trained models
   - Fine-tuning

3. **RAG (Retrieval Augmented Generation)**
   - Vector databases
   - Document retrieval
   - Building RAG systems

4. **Generative AI**
   - GANs basics
   - Image generation
   - Text generation
